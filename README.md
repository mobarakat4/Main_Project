
## Requirements
- PHP 8.2

## Installation
```sh
git clone https://github.com/mobarakat4/Main_Project.git
cd Main_Project
cp .env.example .env
composer install
php artisan key:generate
php artisan mi:f --seed
```
## Run The Project
- php artisan serve
- [in chrome] - http://127.0.0.1:8000/login
 - [username] - user
- [password] - user123
